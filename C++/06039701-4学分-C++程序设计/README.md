对于大多数完全没有编程基础的同学来说，刚开始学习这门课时感到吃力是非常正常的，一方面是 C++ 本身对初学者就不算友好，另一方面是这门课完全印证了那句话——大多数大学里所教的 C++ 只不过是 C with cin cout class。授课老师的水平也是参差不齐，本人的老师居然能在 2025 年做出摆着教室里的电脑不用，在黑板上手写代码讲解这种事；上课的大多数时间只是对着 PPT 照本宣科，习题讲解更是录一个对着 PPT 讲课的视频让大家坐在教室里看。

请不要按照老师要求在电脑上安装 Visual Studio 2010 这种丑陋到让人想吐的软件（这在 Windows 11 上甚至可能无法正常安装），更推荐使用 CLion 或 VS Code（不过既然你都能上 GitHub 了，想必也不用多说）。如果你在学习通上提交作业时发现无法粘贴代码，请深呼吸，看看 [Paster](https://github.com/wed0n/Paster)，或者直接用 AI 写一个模拟粘贴的 python 脚本也行。

对于有电脑的同学来说，统一安排的上机课除了能向助教或授课老师线下提问外没有任何用处。

自 24 级开始，C++ 全面采用机考（包括期中考试和期末考试），经实测可以在编程题中使用 C++ 11 特性（更新的没学过，不懂），这对于自学高手来说无疑是件好事。
